# FullStack E-Commerce

A full-stack e-commerce application built with React (Frontend) and Node.js/Express (Backend) with MongoDB Atlas.

## Tech Stack

- **Frontend:** React, Vite, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **Auth:** JWT
